# 🚀 NovaReward

Welcome to NovaReward!

NovaReward is a modern skill-based rewards platform built using:

- Next.js
- Node.js
- Express.js
- PostgreSQL
- Tailwind CSS

## 🎯 Mission

Create Pakistan's most trusted skill & rewards platform.
